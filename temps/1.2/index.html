My Projects

In the following I want to give the Reader a list of Projects I finished.

  • My first project was a basic to-do-list, which was designed to be a Mischung aus shopping und to-do-list.

  • The second project was a flash-card-style web application in which you were supposed to be able to create projects and to upload flash-cards ment for learning new skills. Additionally I added a document-scanner using CV2.

  • Third project was a Web-Scraping tool in which I used Scrapy. It was to yield prices and if a big Sale was to send me an Email.

  • Fourth was this website in which I wanted to display my skill, projects and combine all past projects into one big project.
