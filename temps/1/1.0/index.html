My Projects

Here I want to display a list of Projects that I finished. The backend of my projects are all django-based. The frontend usually was made using HTML and CSS. With further projects there was more and more JS used.

  • My first project was a to-do-list. Which was created with python in the framework Django.

  • The second project was a flash-card-learning web application, whhich allows you to upload flashcards into different projects. Additionally I added a document-scanner using CV2.

  • Third project was a Web-Scraping tool in which I used Scrapy to yield prices and save them frequently into a SQL-Database. If a big Sale was to send me an Email using EmailJS.

  • Fourth was this personal portfolio website. This project combines all previous projects into one and is supposed to show some frontend skills.
